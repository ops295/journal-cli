# Contributing

Thank you for your interest in contributing to journal-cli! Please follow these guidelines.

Getting started
- Install Go 1.21+.
- Clone the repository and run `make test`.

Running tests
- Run `make test` to execute unit tests.
- Run `make coverage` to generate an HTML coverage report.

Code style
- Follow idiomatic Go formatting: `gofmt` / `gofumpt`.
- Keep functions small and focused.

Pull requests
- Fork, create a branch, implement changes, add tests, and open a PR.
- Use clear commit messages and describe the motivation in the PR body.

Issue reporting
- Create issues with reproduction steps, expected behavior, and logs if available.
